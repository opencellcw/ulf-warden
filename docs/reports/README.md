## Reports

Reports are named with timestamp: `security-audit-report-YYYYMMDD-HHMMSS.md`

View the latest report:
```bash
ls -lt security-audit-report-*.md | head -1
```

## Automation

Reports can be generated automatically:
- After deployments
- As part of CI/CD pipeline
- On a scheduled basis (e.g., weekly)
